# Cybersecurity Awareness Chatbot

A console-based chatbot to educate users about cybersecurity threats.

## Features
- 🎤 Voice greeting on startup
- 🎨 ASCII art logo
- 💬 Interactive chat
- 🔒 Topics: phishing, passwords, malware, social engineering
- ⏱️ Typing effect
- ✅ Input validation

## How to Run
1. Clone the repo
2. Open in Visual Studio 2022
3. Add reference to System.Windows.Forms
4. Place `greeting.wav` in bin/Debug
5. Press F5
